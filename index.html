Game Help
  • This is an implementation of basic Star Wars Role Playing Game using JavaScript.
  • The game uses jQuery for events/page manipulation and JavaScript for the game logic.
  • To play the game:
    • Select a character that you wish to play as
    • Then select the first character that you wish to attack
    • You must continue attacking that character until either you defeat the character or until you lose the game by runningg out of health points.
    • If you won the previous attack, select a new character to attack and repeat until all characters are defeated
  • The git repository for the project is located at https://github.com/semlak/week-4-game/
  • Note that this is not currently working as a responsive web-app, and does not support a mobile device interface
