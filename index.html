Game Help
  • This is an implementation of the game Hangman primarily using JavaScript.
  • All of the words from from Tolkien's Legendarium (Lord of the Rings/Silmarillion)
  • To start a game, press any key
  • During a game, with the webpage having keyboard focus, press any letter, trying to guess the Hangman word
  • The git repository for the project is located at https://github.com/semlak/Hangman-Game/
  • Note that this is not currently working as a responsive web-app, and does not support a mobile device interface
